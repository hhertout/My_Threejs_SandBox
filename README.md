# Three Js SandBox 🎆

Sandbox to learn and practise Three JS.

## ⚒️ Setup 

- Three JS
- Webpack (not config)
- Babel (not config)
- GreenSock
- dat.GUI
- Sass

### 🖥️ Installation 

npm install & you good to go !

### Sass Command

sass --watch public/assets/stylesheets
